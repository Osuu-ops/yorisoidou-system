# 00_DRAFT (RAW / DRAFT lanes)
このディレクトリは「草案（RAW/DRAFT）を物理的に隔離して保管する」ためのレーンです。
Bundled（docs/MEP/MEP_BUNDLE.md）は Adopted（確定マスタ）束ねであり、草案の保管場所を表現しません。
## Lanes
- BUSINESS/ : ビジネス草案（未確定）
- SYSTEM/   : システム草案（未確定）
## Rules (minimum)
- ここに置かれた内容は **未確定** として扱う（Adopted / Bundled の根拠にはならない）
- 確定（Adopted）へ昇格する場合は、所定の Gate / PR フローでカード・マスタ側へ移送する
