## 本リポジトリについて

本リポジトリは、  
**MEP（Meaning Enforcement Platform）** に基づき構築された  
業務自動化プラットフォームである。

本プロジェクトは、

- 思想
- 統治
- 業務仕様
- UI 表現
- システム制御

を明確に分離し、  
**誤解・推測・再解釈を排除した状態**で  
業務を再現可能にすることを目的とする。

## 正の所在（最重要）

本プロジェクトにおける「正」は、  
以下に **一意に定義**されている。

- 思想・統治：`MEP_PROTOCOL.md`
- システム制御：`system_protocol.md`
- UI 意味制約：`UI_PROTOCOL.md`
- 境界統治：`INTERFACE_PROTOCOL.md`
- 業務仕様：`master_spec`

これら以外の文書・実装・挙動は、  
正の根拠とはならない。
