# MEP_OPERATION_NAVIGATOR_SPEC

## 目的
MEPが「止めた」際に、理由を明確化し、
人が迷わず次の一手を選べるようにする。

## 基本原則
- 止まる = 事故防止として正しい挙動
- 正解は提示しない。選択肢を提示する
- 最終判断は人が行う
- すべて日本語で説明する

## 出力フォーマット（固定）
### 停止理由（Fact）
- 該当 Gate / DoD / Scope
- 不足・矛盾・禁止内容

### 直し方の選択肢（Choice）
1. 標準修正（王道）
   - 最小変更でゲートを通す
2. 暫定回避（現場優先）
   - 証跡必須・後で是正
3. 根治策（再発防止）
   - マスタ／テンプレ改善
4. 天才ムーブ（発想転換）
   - 状態・手順・設計そのものを見直す

### 確認ポイント（最大3つ）
- 優先：速度 / 安全 / 再発防止
- 今回は例外か？
- 影響範囲は許容内か？

## 注意
- AIは断定しない
- 推測は明示（Guess）
