# Z_AXIOMS_CANON
（Z フェーズ公理定義）

本書は、Z フェーズが NG を返す
**非解釈・非裁量・機械判定可能な条件**を定義する。

本書は説明を行わない。
本書は評価を行わない。
本書は Yes / No 判定のみを目的とする。

---

## 公理 Z-01：行削除禁止

対象文書において、
**既存行の削除が1行でも検出された場合**、
Z は NG を返す。

---

## 公理 Z-02：行置換禁止

対象文書において、
**同一行番号で内容が異なる行が検出された場合**、
Z は NG を返す。

（意味同一・表現改善・誤字修正は考慮しない）

---

## 公理 Z-03：diff 非経由変更禁止

diff を伴わない変更が検出された場合、
Z は NG を返す。

---

## 公理 Z-04：read-only 領域書込み禁止

以下の read-only 領域に対する書込みが検出された場合、
Z は NG を返す。

- X（最新確定仕様）
- S（最新確定成果物）
- Canon / Protocol / Master 文書

---

## 公理 Z-05：Z NG 後の自動継続禁止

Z が NG を返した後に、
自動処理・自動遷移・自動生成が試みられた場合、
Z は NG を返す。

---

## 公理 Z-06：保存条件違反禁止

Z OK 以外の状態で、
X または S への保存が検出された場合、
Z は NG を返す。

---

## 公理 Z-07：境界違反入力禁止

API 実装時において、
以下の違反が検出された場合、
Z は NG を返す。

- read-only 入力の再生成
- candidate の確定扱い
- 会話文脈を判断根拠に使用

---

## 付記

- 上記いずれか1つでも成立した場合、Z は NG を返す。
- Z は理由説明を行わない。
- Z は修正案を提示しない。
- Z は停止のみを行う。

---
